# SNS Buffer MVP

Buffer API 기반 SNS 예약/발행 MVP 기획과 실행을 위한 문서 모음입니다.

## 범위
- 인스타그램, 스레드 등 3개 채널까지 무료 제공을 가정
- 핵심은 `연결 -> 작성 -> 예약 -> 발행 -> 결과 확인`

## 문서 위치
- `docs/index.md`: 문서 인덱스
- `docs/pages/`: 페이지별 문서
- `docs/frontend/`: 프론트 설계
- `docs/backend/`: 백엔드 설계
- `docs/infra/`: 클라우드/배포
