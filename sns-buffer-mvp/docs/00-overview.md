# 개요

## 문제 정의
개인/소상공인이 여러 SNS 계정을 운영할 때, 글 작성/예약/발행/성과확인을 한곳에서 처리하기 어렵다.

## 제품 비전
Buffer API를 사용해 최소한의 UI로 SNS 발행을 자동화하고, 사용자가 꾸준히 발행할 수 있도록 돕는 MVP.

## 목표
- 10분 내 첫 발행 완료
- 월 1회 이상 재사용
- 인스타그램/스레드 등 3개 채널 무료 제공

## 비목표
- 고급 팀 협업(권한/승인 흐름)
- 대규모 분석 대시보드
- 자체 SNS API 직접 연동(초기에는 Buffer만)

## 주요 페르소나
- 1인 크리에이터
- 소상공인 마케터(혼자 운영)
- 초기 스타트업 운영자

## 가정
- Buffer API로 발행/예약이 가능
- 무료 플랜은 최대 3개 채널
- 웹 앱 중심 MVP

## 열린 질문
- 프로젝트 폴더명은 무엇으로 할지?
- MVP 대상 채널 확정(인스타그램, 스레드, X, 페이스북 등)
- 로그인 방식(이메일/소셜/OAuth만?)
- 과금/플랜 구조(무료/프로/팀)
- 목표 시장(한국/글로벌)
