# 요구사항

## 기능 요구사항
- 회원가입/로그인
- Buffer 계정 연결
- SNS 채널 연결(최대 3개)
- 게시물 작성(텍스트, 이미지, 해시태그 템플릿)
- 예약 발행(일정 선택)
- 예약 목록/캘린더 보기
- 발행 결과 상태 확인(성공/실패/재시도)
- 기본 분석(최근 7일 발행 수, 채널별 발행 수)
- 결제/플랜(무료/유료 전환은 후순위)

## 비기능 요구사항
- 1차 MVP에서는 1초 이내 목록 로딩
- 장애 시 발행 실패 알림(이메일/앱 알림)
- 최소한의 개인정보 저장
- 모든 외부 API 키는 서버 측 관리

## 제약
- SNS 직접 연동 대신 Buffer API 사용
- 무료 플랜은 채널 3개 제한

## 성공 지표
- 첫 발행 완료율 60% 이상
- 7일 리텐션 20% 이상
- 평균 발행 준비 시간 5분 이내
